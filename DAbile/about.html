About: Dispensabile


Dispensabile è un software di gestione dispensa realizzato in ASP.NET Web Forms (C#), ADO.NET MDB e Angular.js


Il software può essere hostato su Microsoft Internet Information Services 8+ oppure self-hosted in un'applicazione Console Windows tramite Open Web Interface for .net & Katana


Angular.js è utilizzato per il two-way binding nell'interfaccia utente.


Il software non presenta autenticazione, si raccomanda quindi l'hosting in rete locale o su server privato con sistemi di autenticazione esterni.
