DispensAbile è un software di gestione dispensa realizzato in ASP.NET Web Forms (C#), MDB e Angular.js

Lavoro al progetto nel tempo libero, quindi gli aggiornamenti non saranno particolarmente costanti, sorry.

In realtà, Angular.js non è utilizzato al momento, ma verrà incluso nelle prossime versioni.

Pubblicherò a breve una nuova versione [1.1.6] che comprenderà:

  1. Web Forms Authentication
  2. Modifica della stampa tramite print-specific css
  3. Animazioni e tooltips per un'utilizzo più semplice e chiaro con Angular e Bootstrap
  4. Aggiunta di un campo Note [LongText]
  5. Probabile conversione del db ad una tecnologia più robusta (SQLite 3 o altro)

Ho sviluppato il software per l'associazione L'Abbraccio-onlus e verrà utilizzato a partire da ottobre, su un server locale IIS8@ Windows Server 2012 R2
